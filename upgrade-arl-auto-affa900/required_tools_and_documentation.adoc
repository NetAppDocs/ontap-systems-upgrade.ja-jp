[role="lead"]
アップグレードを実行するには接地ストラップが必要です。また、アップグレードプロセス中に他のマニュアルを参照する必要があります。

AFF A800をAFF A90またはAFF A70にアップグレードする場合は、100GbEケーブルの長さが1メートル以上であることを確認します。

= コントローラのアップグレードプロセスを自動化する
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
コントローラのアップグレード時に、コントローラは、より新しい、またはより強力なプラットフォームを実行する別のコントローラに交換されます。ここでは一部自動化された手順について説明します。この手順では、ネットワークポートの到達可能性チェックが自動で行われるため、コントローラのアップグレードがさらに簡単になります。

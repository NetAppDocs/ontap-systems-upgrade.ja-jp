= スイッチを永続的に有効にします
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ファブリック内のスイッチを永続的に有効にする必要があります。

スイッチを永続的に有効にします。

* Brocade スイッチの場合は、「 witchCfgPersistentenable 」コマンドを使用します。
+
[listing]
----
FC_switch_A_1:admin> switchCfgPersistentenable
----
* Cisco スイッチの場合は 'no suspend' コマンドを使用します
+
[listing]
----
vsan [vsna #]no suspend
----

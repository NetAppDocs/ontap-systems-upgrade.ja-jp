= ノード 2 を撤去
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
node2 を廃止するには、node2 を適切にシャットダウンし、ラックまたはシャーシから取り外します。

.手順
. 処理を再開します。
+
「システムコントローラの交換が再開」

+
ノードは自動的に停止します。



.次の手順
node2を廃止した後、アップグレード構成に基づいて、以下のコントローラ交換手順のいずれかを選択してください：

* link:replace-node2-affa250-affc250.html["node2 AFF A250 または AFF C250 コントローラモジュールを交換します"]
* link:replace-node2-affa800-affc800.html["node2 AFF A800 または AFF C800 コントローラモジュールを交換します"]
* link:replace-node2-affa220-affa200-affc190-fas2620-fas2720.html["node2 AFF A220、AFF A200、AFF C190、FAS2620、またはFAS2720コントローラモジュールを交換します"]
* link:replace-node2-aff700-fas900.html["node2 AFF A700またはFAS9000コントローラとNVRAMモジュールを交換します"]
